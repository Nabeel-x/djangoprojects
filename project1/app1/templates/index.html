Django Framework

Django is an open source web application framework written in Python programming language, which allows quick developmetn of efficient web applications. It provides built-in features such as Django Admin Interface, default database SQLite3,etc.

Django is based on MVC(Model-View-Controller)pattern

Model : It is the logical data structure behind the entire application and is represented by a database.
